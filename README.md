### Options

Otherwise, you can fill up your terminal with options like an atomic fungus.

*RYM export file*

```
$ rym2discogs --file ./export.csv
$ rym2discogs -f ./export.csv
```

*RYM user ID*

```
rym2discogs --id
rym2discogs -i
```

*Discogs token*

```
rym2discogs --token
rym2discogs -t
```

*Discogs username*

```
rym2discogs --username
rym2discogs -u
```
